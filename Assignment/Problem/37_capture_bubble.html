https://my.newtonschool.co/playground/markup/co3kx3mge00f Event capturing means propagation of event is done from ancestor elements to child element in the DOM while event bubbling means propagation is done from child element to ancestor elements in the DOM. The event capturing occurs followed by event bubbling. If {capture: true} ,event capturing will occur else event bubbling will occur. Both can be prevented by using the stopPropagation() method.
